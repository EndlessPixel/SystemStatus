# Favicon 文件说明

请将您的 favicon.ico 放在本目录。

## 推荐尺寸

- favicon.ico: 16x16, 32x32, 48x48 (多尺寸 ICO 文件)
- 可选: favicon-16x16.png
- 可选: favicon-32x32.png
- 可选: apple-touch-icon.png (180x180)

## 如何添加

1. 生成或获取您的 favicon 文件
2. 命名为 `favicon.ico`
3. 放置到 `public/` 目录
4. 访问 `/favicon.ico` 即可查看

## 生成工具推荐

- https://realfavicongenerator.net/
- https://favicon.io/
- https://www.favicon-generator.org/

## 访问方式

- `/favicon.ico` （自动从 public/ 目录提供）
- `/public/favicon.ico` （通过静态文件访问）
